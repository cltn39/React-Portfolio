### Prerequisites

* npm


### Installation

1. You will need an API key from Nimble Neuron first.
2. Clone the repo
   ```sh
   git clone https://github.com/cltn39/React-Portfolio.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. start the application
   ```sh
   npm start
   ```
